# Desktop Markdown Viewer

Um leitor de arquivos markdown para desktop, feito com Tauri e Rust.

## Funcionalidades

### V0.9.0
- [x] Abertura de arquivos markdown
- [x] Visualização de arquivos markdown
- [x] Side-bar para navegação entre arquivos
- [x] Leitor de mermaid

### V1.0.0
- [ ] Exportar para PDF
- [ ] Sistema para abrir pastas
- [ ] Correção de erros conhecidos v0.9.0


## Erros conhecidos V0.9.0
- [ ] Falta um botão para voltar para o arquivo pai
- [ ] Ao abrir um link de um site, ele redireciona pro site
- [ ] Ao usar um link com sublink, ele não abre o arquivo



# Dependencias

- Tauri
- Rust

cargo tauri dev
